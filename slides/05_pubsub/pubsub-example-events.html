PubSub (LaxarJS) - Beispiel - Events

- Die *konsequente lose Kopplung* ermöglicht eine Verbindung zur *Konfigurationszeit*. - Selbst auferlegt: Statische Verbindung per JSON-Konfiguration. Manchmal unflexibel, erlaubt aber die Anwendungsdynamik statisch zu erfassen. - Verbindungen zwischen Teilnehmern werden über gemeinsame Topics gebildet, die jeweils den entsprechenden "Anschlüssen" Activity- und View-Komponenten zugeordnet sind. - Aktionen können dabei jeweils aus verschiedenen Quellen angestoßen werden (Array-config), Ressourcen haben dagegen immer genau eine *owner*-Activity, hier würde das Array keinen Sinn machen. - Dieses deklarative Format kann von Werkzeugen ausgewertet werden, um einen Überblick über komplexe Anwendungen zu erhalten.
  • JSON-Seitenkonfiguration
  • Deklarative Zuordnung von Kollaborateuren
  • Statisch, aber beliebig erweiterbar (z.B. 2 shopping-carts)

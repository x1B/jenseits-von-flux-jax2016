Redux - Beispiel - Store


import * as redux from 'redux';
import thunk from 'redux-thunk';
import reducer from './reducers';
import { getAllProducts } from './actions';
import { render } from './view';


const createStore = redux.applyMiddleware(thunk)(redux.createStore);
const store = createStore(reducer);

store.dispatch(getAllProducts());

render(store, document.getElementById('redux-app'));
- Store-Implementierung nimmt man nicht selber vor - Stattdessen wird eine Factory-Funktion für generische Stores verwendet - Dadurch sieht man hier Bootstrapping-Code und keine Implementierung - Mit Hilfe von `createStore` wird der Store erzeugt und mit dem Root-Reducer initialisiert - Hier ist jetzt noch die Konfiguration von thunk als Middleware dazwischen - Initial wird nur die Action zum Abholen aller Produkte angestoßen - View wird mit Store und DOM-Knoten zum einhängen gerendert
  • Nur ein generischer Store (Daten-Baum)
  • Einzige Quelle für aktuellen Zustand
  • Reducer für Teilbäume
  • Ein Root-Reducer verteilt Verantwortung
  • thunk wie eben gesehen
